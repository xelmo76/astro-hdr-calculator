📋 Processo Step-by-Step

1. Preparazione
Assicurati che i tre stack siano nella directory di lavoro di Siril
2. Verifica dimensioni
load stack1.fit
load stack2.fit
load stack3.fit
Controlla che abbiano le stesse dimensioni
3. Allineamento
Se necessario:
register stack2.fit stack1.fit
register stack3.fit stack1.fit
4. Pixel Math
Menu: Image Processing → Composition → Pixel Math
Inserisci la formula calcolata sopra
5. Salvataggio
save risultato_HDR
1. HDRComposition
Process → HDRComposition
2. Aggiungi immagini
Carica i tre stack nell'ordine: breve → medio → lungo
3. Parametri
• Mask smoothness: 0.7
• Generate mask: checked
• Auto stretch: checked
4. Esecuzione
Clicca Apply per generare l'HDR automatico
1. Apertura
Apri lo stack con più dettagli come immagine base
2. Livelli
Trascina gli altri due stack come livelli
3. Blend modes
Prova: Screen, Lighten, Overlay, Linear Light
4. Opacità
